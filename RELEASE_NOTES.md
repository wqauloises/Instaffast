## InstafFast v1.0 Release

This is the compiled Android APK ready to download and install.

### 📱 Download
- **instaffast-release.apk** - Ready to install on Android 7.0+

### ✨ Features Included
- Instagram followers & following widget
- Simple dark theme UI
- Manual refresh button
- Profile tap-to-open
- Auto-reboot persistence

### 📋 Installation Steps
1. Download `instaffast-release.apk`
2. Open on your Android device
3. Allow installation from unknown sources
4. Install and open
5. Add widget to home screen
6. Enter your Instagram username
7. Done! ✅

### 🔧 Requirements
- Android 7.0 (API 24) or higher
- Active internet connection
- Public Instagram profile

### 📝 Version Info
- **Version**: 1.0
- **Build Date**: 2026-06-03
- **Target SDK**: 36
- **Min SDK**: 24 (Android 7.0)

### 🐛 Known Issues
- Private Instagram profiles may not work
- Instagram may change their website structure

### 💡 Tips
- Long press home screen → Widgets → InstafFast
- Tap widget to configure with your username
- Use refresh button to update counts
- Remove and re-add widget if it stops working

Enjoy using InstafFast! 🚀
